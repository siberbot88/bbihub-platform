Pertanyaan yang Sering Diajukan

Temukan jawaban untuk pertanyaan umum tentang BBiHub.

Tidak ada biaya setup atau instalasi. BBiHub adalah platform SaaS berbasis cloud yang langsung bisa Anda gunakan setelah registrasi. Semua paket sudah termasuk onboarding dan training gratis.

Data Anda tersimpan aman di server cloud dengan enkripsi SSL/TLS. Kami melakukan backup otomatis setiap hari dan mematuhi standar keamanan industri. Data Anda hanya bisa diakses oleh tim Anda.

Ya, tim kami akan membantu Anda migrasi data dari sistem lama ke BBiHub. Untuk paket Enterprise, kami menyediakan dedicated support untuk memastikan proses migrasi berjalan lancar tanpa kehilangan data.

Kami menyediakan support via WhatsApp, email, dan chat di dashboard. Paket Free mendapat email support, paket Pro mendapat priority support dengan response time 4 jam, dan Enterprise mendapat dedicated account manager.

Anda bisa bayar subscription via transfer bank, kartu kredit, dan e-wallet. Untuk pelanggan Anda, BBiHub terintegrasi dengan payment gateway yang menerima semua metode pembayaran termasuk QRIS.

Ya, paket Enterprise dirancang khusus untuk bengkel dengan multi cabang. Anda bisa kelola semua cabang dalam satu dashboard, lihat laporan konsolidasi, dan tetap bisa monitor performa per cabang.

Anda bisa cancel subscription kapan saja tanpa penalti. Data Anda akan tetap tersimpan selama 30 hari setelah cancel, dan Anda bisa export semua data sebelum periode berakhir.

Ya, paket Pro tersedia free trial 14 hari dengan full access ke semua fitur. Tidak perlu kartu kredit untuk memulai trial. Setelah trial berakhir, Anda bisa pilih untuk upgrade atau tetap di paket Free.
