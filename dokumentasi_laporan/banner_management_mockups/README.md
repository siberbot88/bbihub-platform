# Banner Management System - Mockups untuk Laporan

Folder ini berisi mockup dan dokumentasi visual untuk sistem manajemen banner BBI HUB Platform.

## Daftar File

### 1. UI Grid Layout (`01_ui_grid_layout.png`)
Desain UI baru dengan grid/card-based layout untuk manajemen banner web dashboard.
- Menggantikan tampilan tabel dengan card grid
- Organized per placement slot
- Heroicons untuk semua aksi

### 2. Size Guide (`02_size_guide.png`)
Panduan ukuran rekomendasi untuk setiap jenis banner:
- **Homepage Admin - Banner Promo**: 1080 x 1080 px (Square 1:1)
- **Homepage Admin - Karakter**: 339 x 339 px (Square 1:1)
- **Owner Dashboard - Banner**: 1024 x 512 px (2:1)
- **Website Landing - Hero**: 1920 x 600 px (16:5)
- **Website Landing - Sub Banner**: 400 x 300 px (4:3)

### 3. Empty State (`03_empty_state.png`)
Mockup tampilan slot kosong:
- Dashed border dengan upload icon
- Menampilkan ukuran rekomendasi
- Call-to-action upload button

### 4. Filled State (`04_filled_state.png`)
Mockup tampilan slot yang sudah terisi:
- Preview banner dengan overlay
- Status badge (Active/Draft/Expired)
- Action buttons dengan heroicons (View/Edit/Delete)

### 5-7. Current State Screenshots
Screenshot dari implementasi sebelumnya untuk perbandingan:
- `05_current_web_dashboard.png`: Web dashboard dengan tabel
- `06_current_mobile_banner.png`: Carousel banner di mobile
- `07_current_mobile_homepage.png`: Homepage admin dengan karakter

## Tujuan Penggunaan

Mockup-mockup ini dibuat untuk:
1. Dokumentasi laporan proyek
2. Presentasi desain UI/UX
3. Perbandingan before/after implementasi
4. Referensi ukuran banner untuk konten kreator
