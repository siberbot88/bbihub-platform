API Documentation

All endpoints are versioned under /api/v1. Authentication uses Laravel Sanctum tokens.

Authentication

Register

POST /api/v1/auth/register

{
    "name": "John Doe",
    "email": "john@example.com",
    "password": "secret",
    "password_confirmation": "secret"
}

Login

POST /api/v1/auth/login

{
    "email": "john@example.com",
    "password": "secret"
}

Logout

POST /api/v1/auth/logout (requires Bearer token)

Current User

GET /api/v1/auth/user (requires Bearer token)

Change Password

POST /api/v1/auth/change-password

Owner (role:owner)

Workshops

POST /api/v1/owners/workshops

PUT /api/v1/owners/workshops/{workshop}

Workshop Documents

POST /api/v1/owners/documents

Employees

GET /api/v1/owners/employee

POST /api/v1/owners/employee

GET /api/v1/owners/employee/{employee}

PUT /api/v1/owners/employee/{employee}

DELETE /api/v1/owners/employee/{employee}

PATCH /api/v1/owners/employee/{employee}/status
