Installation

Step-by-step guide to set up the BBI HUB backend environment.

Prerequisites

  • PHP >= 8.2
  • Composer 2.x
  • MySQL 8.0
  • Node.js & NPM

Setup Project

1

Clone Repository

git clone https://github.com/your-repo/bbihub-backend.git
cd bbihub-backend
2

Install Dependencies

composer install
npm install
3

Environment Configuration

cp .env.example .env
php artisan key:generate

Database Setup

Run migrations and seeders to populate initial data.

php artisan migrate --seed

Default Credentials

Superadmin superadmin@bbihub.com
Password password

Running Application

Backend Server

php artisan serve

Asset Compilation

npm run dev
