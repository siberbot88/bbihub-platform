Web Dashboard Documentation

Comprehensive guide to the BBI HUB admin web dashboard, covering UI components, navigation, and key functionalities.

Dashboard Overview

The dashboard provides real‑time insights, workshop management tools, and user administration capabilities. It is built with Tailwind CSS, Alpine.js and follows the dark theme used across the documentation.

  • Responsive layout for desktop and tablet screens.
  • Quick access panels for workshops, employees, and vouchers.
  • Analytics charts powered by Chart.js.

Business Intelligence (EIS)

Executive Dashboard

A high-level view for decision makers featuring:

  • Real-time KPI Scorecards (Revenue, Active Users).
  • Automated Churn Risk Detection.
  • Platform Health Monitoring (MRR Forecast).

Predictive Analytics

AI-driven insights powered by the Platform Intelligence Service:

  • Churn Risk: Identifies workshops with declining activity (>50% drop).
  • Upsell Spotter: Detects high-volume free users ready for Premium.
  • Revenue Forecast: Linear regression models for next-month MRR.

Automated Reporting

Generate professional PDF reports with a single click. Includes AI-written executive summaries, segmentation charts, and actionable strategic recommendations. Optimized for A4 printing.

Key Sections

User Management

Create, edit, and assign roles (owner, admin, mechanic). Includes password reset flow and activity logs.

Workshop Management

Add new workshops, configure services, upload documents, and view performance metrics.

Voucher Administration

Generate, edit, and track usage of promotional vouchers across workshops.

Analytics & Reports

Visual dashboards for revenue, service popularity, and employee performance.
